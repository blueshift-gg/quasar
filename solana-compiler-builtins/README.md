# solana-compiler-builtins

Compiler runtime intrinsics required by Quasar programs built for Solana SBF.
This `no_std` crate supports the framework; most programs get it through
[`quasar-lang`](https://crates.io/crates/quasar-lang).

- [Quasar guide](https://quasar-lang.com/docs)
- [API documentation](https://docs.rs/solana-compiler-builtins/0.1.0)
