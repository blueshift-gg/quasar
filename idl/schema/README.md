# quasar-idl-schema

The versioned Rust data model for Quasar IDL JSON documents. IDL generation,
proc-macro expansion, and generated-client tooling use the same model.

- [IDL guide](https://quasar-lang.com/docs/core-concepts/idl)
- [API documentation](https://docs.rs/quasar-idl-schema/0.1.0)
