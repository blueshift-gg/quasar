# quasar-idl

IDL assembly and validation for Quasar programs, including deterministic schema
generation and discriminator-collision checks. This crate powers the Quasar CLI
IDL workflow.

- [IDL guide](https://quasar-lang.com/docs/core-concepts/idl)
- [API documentation](https://docs.rs/quasar-idl/0.1.0)
