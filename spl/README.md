# quasar-spl

Zero-copy account types and CPI helpers for the SPL Token and Token-2022
programs. They integrate with `quasar-lang` account validation.

- [SPL Token guide](https://quasar-lang.com/docs/spl-tokens/token-program)
- [API documentation](https://docs.rs/quasar-spl/0.1.0)
