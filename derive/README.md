# quasar-derive

Procedural macros for Quasar programs, accounts, instructions, events, and
errors. The [`quasar-lang`](https://crates.io/crates/quasar-lang) prelude
re-exports the public macros.

- [Quasar guide](https://quasar-lang.com/docs)
- [API documentation](https://docs.rs/quasar-derive/0.1.0)
